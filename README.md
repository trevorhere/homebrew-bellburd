# homebrew-bellburd

Private Homebrew tap for the Bellburd CLI.

## Install

```bash
brew tap trevorhere/homebrew-bellburd
brew install bellburd
```

## Updating

1. Build a new CLI tarball and publish it in the main repo release.
2. Update `Formula/bellburd.rb` with the new URL + SHA256.
3. Commit and push to `main`.
